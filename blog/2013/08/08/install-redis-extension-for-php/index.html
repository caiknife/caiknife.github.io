安装phpredis扩展

Aug 8th, 2013

Posted by CaiKnife Aug 8th, 2013 linux, php, redis, ubuntu

redis官方推荐的PHP扩展有两个——Predis和phpredis。phpredis是使用 C 编写的 PHP module ，速度应该会比较快，而且 CakePHP 使用的 redis 库默认就是 phpredis ，先安装这个试试看。

之后phpinfo一下就能看到phpredis的扩展信息。

redis
Redis Support enabled
Redis Version 2.2.3
